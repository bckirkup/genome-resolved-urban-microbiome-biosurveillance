# R²DI: Predictive Intelligence Architecture (Bioinformatics 2025)

R²DI (Resolution–Responsiveness–Deployability–Intelligence) is the predictive intelligence architecture introduced in our Bioinformatics 2025 manuscript. R²DI is not a separate pipeline; 
it is the conceptual and computational framework that organizes GRUMB’s predictive modules into a scalable and extensible architecture.

R²DI formalizes:
- **simulation-driven ecological blending**
- **entropy-based uncertainty quantification**
- **Minimal Detectable Contamination (MDC) analysis**
- **dominance hierarchy modeling**
- **cross-system resilience and tipping-point analysis**
- **multi-omic extensibility**

GRUMB v2.0 implements these predictive modules for genome-resolved metagenomics.  
R²DI provides the generalized abstraction that enables these methods to scale to multi-country, multi-environment, and multi-omic settings.

All predictive simulations used in the Bioinformatics R²DI manuscript are implemented in this folder.
